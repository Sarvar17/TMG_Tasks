	Сперва прочтем наш файл (путь, т.е path, к которому = “batch07-de.loc”) с File.ReadAllText(path).
Далее создаем массив данных array через функцию Split
с параметрами (‘{’, ’}’, ’:’, ’[’, ‘]’, ‘”’, ‘”’, ‘ ’, “Text”, StringSplitOptions.RemoveEmptyEntries)
делим полученный текст (можно еще воспользоватья c Regex’ом). Далее создаем класс Data, со свойствами Source,
Translation и Key. И в List<Data> data записываем наши данные (через for пробегаемся по массиву и через if
определяем тип данных, к примеру:

For (int i = 0; i < array.Length; i++) {
	if (array[i] = “Source”) { 
		data.Add(new Data(array[i+1], array[i+3], [i+5]));
		i+=5;
	}
}

	Задача А. Подключим EPPlus чтобы создать экзель файл. Чтобы было удобно пользователю для каждого файла будет
свой лист sheet.  Для пользователя будет неудобно если данные будут в столбец, поэтому будем заполнять таблицу
в строчку. В первых трех строках будет шаблон (“Source”, “Translation”, “Key”). Дальше будем таким образом добавлять
элементы с листа data (sheet.Cells[row, column].Value где column всегда равен единице). Стилизация: для размера ячеек
будем использовать автофит с пакета EPPlus (AutoFitColumns()), для выравнивания ExcelHorizontalAlignment.Center;
Стиль текста у шаблона будет Bold (Style.Font.Bold).
